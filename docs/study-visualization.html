Java Swing Study Map

MODO 코드 구조 한눈에 보기

이 문서는 현재 프로젝트 코드를 기준으로 UI, Service, Storage, Model의 역할과 데이터 흐름을 공부하기 쉽게 시각화한 자료입니다.

UI 계층 ui.* 패키지가 화면을 만들고 버튼 클릭 같은 사용자 입력을 받습니다.
Service 계층 TodoService, PomodoroService가 실제 기능 규칙을 담당합니다.
Storage 계층 DataManager가 data/ 폴더의 txt 파일을 읽고 씁니다.
Model 계층 Todo, PomodoroStats가 앱에서 다루는 데이터 모양을 정의합니다.

4. 타이머 완료 흐름

타이머에서 가장 중요한 흐름입니다. 시간이 0초가 되면 Service가 통계를 바꾸고, DataManager가 파일에 저장하고, MainFrame이 화면 통계를 새로고침합니다.

1. START TimerPanel.toggleTimer()가 swingTimer.start()를 호출합니다.
2. 1초마다 tick remainingSeconds를 줄이고 화면 시간을 갱신합니다.
3. 완료 기록 PomodoroService.completePomodoro()가 완료 횟수와 집중 시간을 증가시킵니다.
4. 파일 저장 DataManager.saveStats()가 pomodoro_stats.txt에 씁니다.
5. 화면 갱신 onStatsChanged.run()을 통해 MainFrame.refreshStats()가 호출됩니다. 
sequenceDiagram
    actor User as 사용자
    participant TimerPanel
    participant PomodoroService
    participant PomodoroStats
    participant DataManager
    participant MainFrame

    User->>TimerPanel: START 클릭
    TimerPanel->>TimerPanel: swingTimer.start()
    loop 1초마다
      TimerPanel->>TimerPanel: tick(), remainingSeconds 감소
    end
    TimerPanel->>PomodoroService: completePomodoro(focusMinutes)
    PomodoroService->>PomodoroStats: addFocusSession(minutes)
    PomodoroService->>DataManager: saveStats(todayStats)
    DataManager->>DataManager: data/pomodoro_stats.txt 저장
    TimerPanel->>MainFrame: onStatsChanged.run()
    MainFrame->>MainFrame: lblTotalTime, lblPomodoroCount 갱신
    TimerPanel-->>User: 완료 다이얼로그 표시

6. 저장 파일 지도

이 프로젝트는 DB 없이 txt 파일로 저장합니다. 그래서 파일 포맷을 이해하면 디버깅이 훨씬 쉬워집니다. 

flowchart LR
    DataManager["DataManager"] --> TodoFile["data/todo_list.txt
title|completed|targetDate"]
    DataManager --> StatsFile["data/pomodoro_stats.txt
date|completed|cancelled|totalMinutes"]
    DataManager --> GoalFile["data/daily_goal.txt
date|goal"]
    DataManager --> MemoFile["data/memo.txt
date|memo with <NL>"]

    TodoFile --> TodoService["TodoService 시작 시 loadTodos()"]
    StatsFile --> PomodoroService["PomodoroService 시작 시 loadTodayStats()"]
    StatsFile --> CalendarMemoPanel["CalendarMemoPanel loadAllStats()"]
    GoalFile --> MainFrame["MainFrame 생성자 loadTodayGoal()"]
파일 읽는 메서드 쓰는 메서드 포맷
data/todo_list.txt loadTodos() saveTodos() title|completed|targetDate
data/pomodoro_stats.txt loadTodayStats(), loadAllStats() saveStats() date|completed|cancelled|totalMinutes
data/daily_goal.txt loadTodayGoal() saveTodayGoal() date|goal
data/memo.txt loadTodayMemo() saveTodayMemo() date|memo, 줄바꿈은 <NL>
